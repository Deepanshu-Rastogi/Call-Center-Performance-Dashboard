# Call Center Workforce Performance Dashboard

I developed this operational productivity dashboard using Advanced Excel to track and evaluate call center agent performance. The project focuses on key workforce metrics like total working hours, active engagement, and overall work efficiency to help management optimize operations.

## Key Features & Insights:
- **Employee Working Hours Analysis:** A dynamic column chart tracking the total logged hours for agents (e.g., Ashok Kumar, Manoj Kumar, Rajesh Kumar, Santosh Kumar, Sunil Kumar) to monitor scheduling and availability.
- **Active Time with Customers:** A pie chart breakdown illustrating the percentage of time agents spend actively assisting customers versus idle hours, ensuring better resource utilization.
- **Work Efficiency Metrics:** A structured line chart mapping out agent-wise efficiency rankings based on call targets achieved and time management.
- **Interactive Pivot Reports:** Integrated dynamic headers and automated tables that update agent metrics seamlessly based on data filtering.

## Tech Stack & Features Used:
- **Microsoft Excel** (Data Cleaning & Productivity Metrics Calculation)
- **Pivot Tables & Advanced Sorting** (For aggregating agent-wise shift timelines)
- **Data Visualization** (Customized Line, Pie, and Clustered Column Charts for corporate reporting)
- **Time Formatting Formulas** (To properly handle and calculate `HH:MM:SS` duration formats)
